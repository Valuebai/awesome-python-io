version:Z855_Auto_Run版本：优化容易跑偏的问题

功能：实现Z855手机的功能自动化测试，循环点击发送1-1和群聊的各类消息，无论在不在线
使用：
1. Z855的手机必须root、remount成功，执行脚本过程中手机必须连接电脑
2. 手机至少有几个测试素材


:: 请确认上面的操作完成
::============================================================

1、修改Z855_1_1chat.bat中的号码
::============================================================
::设置第一次发送的1-1号码
if %Flag11% equ 0 (set User_1=%User27%)

::设置第二次发送的1-1号码
if %Flag11% equ 1 (set User_1=%User22%)


2、修改群聊Z855_group_chat.bat中的号码：
::============================================================
::设置第一次发送的群聊号码
if %Flag11% equ 0 (set User_1=%User22%&set User_2=%User27%)

::设置第二次发送的群聊号码
if %Flag11% equ 1 (set User_1=%User24%&set User_2=%User22%)

3、直接执行#1#Z982_Run.bat即可运行1-1和群聊，再关闭所有窗口，执行#2#After_Run_Pull_logs_ToSave.bat和#2#ZTE_AndroidAutocopy.bat进行人工分析和检查UI（如果想单独运行1-1或者群聊，双击对应的脚本即可）
